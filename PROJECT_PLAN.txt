================================================================================
mLab AI Support Platform - 6 Developer Team Breakdown & 3-Week Sprint Plan
================================================================================

OVERVIEW:
This document breaks down the mLab AI Support Platform into 6 distinct pieces,
each assigned to a different developer. The project is built over 3 weeks with
clear deliverables for each week.

================================================================================
6 DEVELOPER ROLES & RESPONSIBILITIES
================================================================================

1. FRONTEND DEVELOPER - Website & Landing Page
   Focus: mLab website mockup, navigation, hero section, branding
   Tech: React, Tailwind CSS, UI/UX design

2. FRONTEND DEVELOPER - Chat Widget UI
   Focus: Chat widget component, message bubbles, animations, styling
   Tech: React, Tailwind CSS, component state management

3. FRONTEND DEVELOPER - Escalation Form
   Focus: Escalation form component, validation, POPIA compliance UI
   Tech: React, form handling, validation

4. BACKEND DEVELOPER - Data Service Layer
   Focus: Firebase service, knowledge base management, data persistence
   Tech: TypeScript, data structures, service architecture

5. BACKEND DEVELOPER - AI Integration Service
   Focus: LLM provider service, API integration, response generation
   Tech: TypeScript, Google Gemini API, async operations

6. FULL STACK DEVELOPER - App Integration & Structure
   Focus: App.tsx, routing, type definitions, constants, connecting all pieces
   Tech: React, TypeScript, project architecture
